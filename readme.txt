# Tubes-Logkom
# Penyelamatan Donatello
> Role Playing Game using Prolog

## Technologies
Menggunakan GNU Prolog 1.4.5

## Setup
1. Install GNU Prolog 1.4.5
   Link: http://www.gprolog.org/
2. Buka folder game pada terminal dan masukkan command pada Linux
    Command : gprolog
        GNU Prolog 1.4.5 (64 bits)
        Compiled Aug 20 2018, 15:27:00 with clang
        By Daniel Diaz
        Copyright (C) 1999-2018 Daniel Diaz
        | ?- ['main.pl'].

3. Mulai bermain dengan 
   Command: ```start.```
